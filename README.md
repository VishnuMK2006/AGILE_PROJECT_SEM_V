# B2EIT Arcade Zone

## üöÄ Getting Started

### Prerequisites

- Python
- Flask

### Installation

1.  **Clone the repository** navigate to the project directory.

2.  **Install dependencies**:

    ```bash
    pip install -r requirements.txt
    ```

3.  **Run the application**:

    ```bash
    python app.py
    ```

4.  **Access the Arcade**:
    Open your web browser and navigate to:
    `http://localhost:5000`
